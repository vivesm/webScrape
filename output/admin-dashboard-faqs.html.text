Operations hub dashboard FAQs
Get answers to frequently asked Operations hub dashboard questions.
What is the Operations hub dashboard?
The Operations hub dashboard provides real-time insights into recipes and connections in your Workato workspace. It offers key insights, analytics, and actionable items to help you resolve issues efficiently.
Learn more about the
Operations hub dashboard
.
How can recipe jobs on the Operations hub dashboard relate to business metrics?
Recipe jobs in the Operations hub dashboard can correlate with business metrics, such as the number of new customers added to your CRM daily or the number of processed invoices. This correlation helps users understand the impact automation has on their business.
Learn more about the
Operations hub dashboard
.
What benefits does the Operations hub dashboard provide for managing recipe issues?
The Operations hub dashboard offers a comprehensive view of recipe health within a workspace. This allows you to address errors efficiently and focus on critical issues.
Learn more about the
Operations hub dashboard
.
How can I customize the Jobs graph on the Operations hub dashboard?
You can customize the jobs graph using filter tools. Modify the graph display using parameters like key folders or specific recipes. This provides flexibility in analyzing and visualizing data.
Learn how to customize the
Jobs graph
.
How does the Recipe details table complement the Jobs graph?
The Recipe details table provides a detailed breakdown of the Jobs graph, offering information on successful jobs, failed jobs, tasks used, and the latest job descriptions for individual recipes.
Learn more about the
Recipe details table
.
Who can use the Activity audit log feature, and how can it be accessed?
The Activity audit log is available to direct customers on specific pricing plans. It is a default feature for embedded partners and their end customers. Users with Admin roles or Activity audit privileges can view the logs.
Learn more about the
Activity audit log
.
What is the concept of zero retention in Workato recipes, and how does it affect audit log streaming?
For recipes with zero retention, Workato sends audit logs without job input and output data. Retry mechanisms apply, similar to other logs, with logs stored in a cache for retry and removed after seven days.
Learn more about
zero retention recipes
.
