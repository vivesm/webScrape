AI by Workato
#
AI by Workato connector - Analyze text action
FEATURE AVAILABILITY
AI by Workato is available for all direct customers across all Workato data centers, but isn’t available to Embedded partners or their customers. All data is sent to OpenAI's servers in the US. You can opt-in to this feature by agreeing to our
AI feature addendum
(opens new window)
. Contact your Customer Success representative to learn more.
Use the analyze text action to understand specific information from a given piece of text. This action facilitates knowledge retrieval tasks and prioritizes the importance of the provided input. If there is no answer found in the text, the action outputs an empty datapill.
Some common use cases include content summarization and sentiment analysis. A user can generate action items from a meeting transcript, title and hints for documents, and even descriptions for technical briefs.
#
Input
Input field
Description
Source text
Provide the source text you plan to analyze.
Instruction
Provide instructions on how to analyze your source text. For example, this could be an analytic technique or question you want answered.
#
Output
Output
Description
Analysis result
The result of the action. The output is determined by the instructions you provide.
Remaining calls
The number of remaining calls Workato can execute out of the overall daily limit (100 calls).
Consumed calls
The number of actions completed out of the overall daily limit (100 calls).
Reset time
The time the daily limit resets.
