Intercom
#
Intercom - Archive users action
The
Archive users
action allows you to archive users in Intercom. This action archives users based on a list of email addresses or user IDs that you provide.
Intercom - Archive users action
#
Input
Parallel
Specify the number of parallel requests to process. The minimum value is 1, and the maximum is 10.
Requests per minute
Set the target request rate per minute. The rate is calculated on a 10-second interval basis. For detailed rate limits, refer to the
Intercom API rate limit documentation
(opens new window)
. Setting this field to zero, or leaving it empty, removes rate limiting.
Users
Provide the list of users to archive. You can specify users by their email address or user ID.
