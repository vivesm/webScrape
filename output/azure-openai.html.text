#
Azure OpenAI Connector
Azure OpenAI
(opens new window)
is a collaboration between Microsoft Azure and
OpenAI
(opens new window)
, the artificial intelligence research organization behind the popular ChatGPT, which interacts in a conversational way. Azure OpenAI is a cloud based platform that enables users to build and deploy AI models.
Beyond that, it has multiple models you can leverage for editing text, generating images, and even classification. You can fine-tune and create your own models, exposing them internally using Azure OpenAI.
#
API Version
The Azure OpenAI connector uses the
Azure OpenAI Service REST API
(opens new window)
.
#
How to connect to Azure OpenAI on Workato
The Azure OpenAI connector uses an API key for authentication. Learn how to generate your API key by following the instructions in
Azure OpenAI's documentation
(opens new window)
. We recommend that you store the API key safely, as it is only displayed one time.
