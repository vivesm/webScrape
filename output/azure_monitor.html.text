#
Azure Monitor
Azure Monitor
(opens new window)
, is Azure's logging service that allows you to collect logs from a variety of different sources and allows you to query these logs in Azure. With Workato, you'll be able to use the Azure Monitor connector to send any custom logs from your recipes to Azure monitor for logging purposes.
You may also use the Azure Monitor connection to stream any audit logs from Workato using our
Audit log streaming
.

#
How to connect to Azure Monitor on Workato
Field
Description
Connection name
Give this connection a unique name so you can reuse it in other recipes.
Workspace ID
The ID of your log analytics workspace. See below for more details.
Shared key
The primary or secondary key of your log analytics workspace. See below for more details.
#
How to get your Workspace ID and Shared Key
1
Head over to your the log analytics workspace where you want to send logs to.
2
Select "Agents management".
3
You can find your Workspace ID and shared keys here. You may use your primary or secondary key as your shared key.
