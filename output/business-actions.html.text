#
ビジネスアクション
AIを使用してビジネスに取り組むためのスタートアップを提供するために、共通のビジネスアクションを作成しました。これらのアクションは事前に設定されており、自分自身でプロンプトを作成したり調整する必要なく、すぐにOpenAIのモデルから価値を得ることができます。
これらのアクションは、OpenAIのGPT-3.5-Turboモデルを使用し、
conversations endpoint
を活用しています。Workatoは以下のアクションをサポートしています：
テキストの分析
テキストの分類
メールの下書き
テキストの解析
テキストの翻訳
#
テキストの分析
テキストの分析アクション
テキストの分析アクションを使用すると、与えられたテキストから特定の情報を簡単に把握することができます。これは、知識の検索タスクや提供された入力の重要性の優先順位付けを支援することを目的としています。テキスト内に回答が見つからない場合、アクションは空のデータピルを出力します。
一般的な使用例には、コンテンツの要約や感情分析があります。ユーザーは、会議のトランスクリプトからアクションアイテムを素早く生成したり、ドキュメントのタイトルやヒントからアクションアイテムを生成したり、技術的なブリーフの説明やヒントからアクションアイテムを生成したりすることができます。
#
テキストの分類
テキストの分類アクション
テキストの分類アクションを使用すると、事前に定義されたカテゴリに基づいてテキストデータを自動的に分類または整理することができます。一般的なカテゴリの場合、ルールを省略することもできますが、一般的には分類を支援するために明確なルールの説明を設定することをおすすめします。
このアクションを使用すると、ビジネスプロセスを効率化するために、受信したテキスト情報を効果的にソート、グループ化、またはルーティングすることができます。例えば、カスタマーサポートチケットのルーティング、コンテンツのタグ付け、スパムフィルタリング、製品の分類などのワークフローを補完することができます。
#
メールの下書き
メールの下書きアクション
メールの下書きアクションを使用すると、コードを書かずにビジュアルインターフェースを使用してパーソナライズされたメールを作成することができます。このアクションにより、個人やグループに対してカスタマイズされたメールを簡単に作成して送信することができ、コミュニケーションのワークフローを自動化することができます。
このアクションを使用すると、マーケティングや営業のワークフローでパーソナライズされたメールの下書きを作成することができます。セールスファネルのステージに応じたさまざまなタイプのメールでリードナーチャリングを行うことができ、過去のメールや他のデータのユーザー固有のコンテキストを追加することができます。このアクションは、マーケティングキャンペーンやイベントリマインダーなど、さまざまなキャンペーンやイベントにわたる最小限のカスタマイズが必要なマーケティング資産にも役立ちます。
#
テキストの解析
テキストの解析アクション
テキストの解析アクションを使用すると、コーディングのスキルを必要とせずに、非構造化テキストから構造化データを抽出することができます。テキストドキュメント、メール、またはその他のテキストソースから特定の情報を識別して抽出することができます。この情報には、名前、住所、日付、製品の詳細などが含まれます。
このアクションをリードジェネレーションのワークフローの一部として使用することで、受信したテキストメッセージ、チャットの会話、またはソーシャルメディアの投稿から顧客情報を抽出し、リードをキャプチャして分類することができます。
#
テキストの翻訳
テキストの翻訳アクション
テキストの翻訳アクションを使用すると、効率的なコミュニケーションを可能にするために、テキストを一つの言語から別の言語に自動的に翻訳することができます。このアクションを使用することで、翻訳プロセスを自動化し、ワークフローに多言語の機能を簡単に組み込むことができます。
ユーザーは、さまざまな言語でのサポートチケットや問い合わせを共通の言語に翻訳することができ、ウェブサイトやドキュメントのためのローカライズされたコンテンツを生成することができます。また、マーケティング資産を簡単に言語ごとに変更するのにも役立ちます。
