#
AI by Workato - Categorize text action
FEATURE AVAILABILITY
AI by Workato is available for all direct customers across all Workato data centers, but isn’t available to Embedded partners or their customers. All data is sent to OpenAI's servers in the US. You can opt-in to this feature by agreeing to our
AI feature addendum
(opens new window)
. Contact your Customer Success representative to learn more.
categorize text actionは、事前に定義されたカテゴリに基づいてテキストデータを自動的に分類または整理することを可能にします。一般的なカテゴリの場合、ルールを省略することができますが、一般的には分類を助けるために明確なルールの説明を設定することをお勧めします。
このアクションにより、受信テキスト情報を効率的に整理、グループ化、またはルーティングして、ビジネスプロセスを合理化することができます。強化できる可能性のあるワークフローには、カスタマーサポートチケットのルーティング、コンテンツのタグ付け、スパムフィルタリング、製品分類などがあります。
#
Input
Input field
Description
Text
コネクタが分類する予定のテキストを提供します。
List of categories
テキストを分類するためのカテゴリのリストを作成します。各カテゴリが何を表すかを分類するのに役立つ追加の詳細を提供するためのルールを作成します。
#
Output
Output
Description
Best match category
このアクションは、入力テキストに最も適したカテゴリの1つを選択します。出力データピルには、最適なカテゴリの値が含まれます。定義したカテゴリがソーステキストに見つからない場合はエラーが表示されます。オプションとして「なし」を設定する場合は、明示的に設定する必要があります。
Remaining calls
1日の全体制限（100回）のうち、Workatoが実行可能な残りの呼び出し回数。
Consumed calls
1日の全体制限（100回）のうち、既に完了したアクションの回数。
Reset time
1日の制限がリセットされる時刻。
