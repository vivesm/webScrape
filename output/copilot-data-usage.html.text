Copilots - Data usage
We prioritize your privacy and data security, selectively retrieving contextual data to ensure optimal Copilot performance. Copilot only uses data directly related to the recipes you create within Workato. This does not include any third-party application data or job logs.
Specifically, recipe data is defined as recipe steps, input fields, and apps used in your recipes.
#
Data usage by feature
The following is a summary of the data used for its various features:

Data used

Data about your current recipe and input fields.

Formula Copilot

User prompts for connector actions, API documentation, and metadata about the connectors being developed.
See example Connector Copilot data usage

Data from the input fields to create the proper output for actions such as text analysis, categorization, email drafting, parsing, and translation.

Examples by feature
The following examples demonstrate the data usage for each Copilots feature:
#
Recipe Copilot: Recipe generation
Recipe Copilot uses your prompt and the connection you select to generate a new recipe
#
Recipe Copilot: Add next step
Recipe Copilot uses your prompt, the current recipe data, and the selected connection to suggest adding a new step
#
Docs Copilot
Docs Copilot uses the recipe data and data from called functions to generate a recipe description
#
Formula Copilot
Formula Copilot uses your prompt, field structure metadata, and recipe data to generate a formula
#
Mapper Copilot
Mapper Copilot uses your prompt, field structure metadata, and recipe data to generate a text input suggestion
#
Connector Copilot
Connector Copilot uses your prompt, API documentation, and metadata to assist in creating and refining custom connectors
#
AI by Workato
AI by Workato uses data from input fields to create output for actions such as text analysis, categorization, email drafting, parsing, and translation
