#
JumpCloud - Create association action
The
Create association
action creates an association between two objects you specify in JumpCloud.
#
Input
Field
Description
Association type
Select the object types you plan to create an association between.
ADDITIONAL INPUT FIELDS
The recipe editor generates additional input fields based on the
Association type
you select. Use these fields to enter the JumpCloud IDs of the objects you plan to associate. Refer to JumpCloud's
Retrieve Object IDs from the Admin Portal
(opens new window)
guide to retrieve these IDs.
#
Output
Field
Description
Status
Displays the status of the API request for the
Create association
action.
