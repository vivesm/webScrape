Epoch
Convert a date and time to the number of seconds that have elapsed since January 1, 1970 (midnight UTC/GMT) not counting leap seconds.
See 
UNIX
 to use the number of milliseconds elapsed instead of seconds.

Definition
Type
Required

TRUE

Date and time in epoch format. Seconds are counted since January 1, 1970, midnight UTC/GMT, not counting leap seconds.
