テキスト編集アクション
提供された入力、指示、およびパラメータに対してテキストを編集します。このアクションは、さまざまなユースケースでワークフロー全体で使用することができます：
提案/ピッチの生成:
セールスチームは、提案書、見積もり、その他のセールス資料の品質を向上させるために、テキスト編集機能を使用することができます。このアクションは、セールスドキュメントを生成し、内容を明確かつ簡潔にするために自動的に編集するために使用することができます。
ブレインストーミング:
マーケティングチームは、メッセージングやコンテンツの品質を向上させるために、テキスト編集機能を使用することができます。マーケティング資料を生成し、文法的に正確で魅力的なコンテンツに自動的に編集するためのワークフローで使用することができます。
ドキュメンテーションの生成:
このアクションは、リリースノートからドキュメンテーションを生成するために使用することができます。これにより、内部および外部のコミュニケーションが迅速化されます。

モデルが入力テキストを編集するための具体的な指示を提供します。

完了の多様性を制御するために0から1の値を入力します。値が高いほど、より多様な応答が生成されます。これと温度の両方を使用することはお勧めしません。詳細は

温度
完了のランダム性を制御するために0から2の値を入力します。値が高いほど、出力はよりランダムになります。一方、値が低いほど、より焦点を絞った決定論的な出力になります。これとtop pの両方を使用することはお勧めしません。詳細は

プロンプトに対して生成する編集の数です。最小値は1、最大値は20です。
