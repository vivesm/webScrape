SHA256
The 
SHA256
 function card hashes input text using the SHA-256 hash function and returns the resulting hashed text.
Input
Field
Definition
Type
Required
data
Text string to be hashed.
String
TRUE
Output
Field
Definition
Type
output
                            Encoded string of your input data.
The output  is in Base64 binary to ASCII text encoding.
String
You can use the result of the 
output
 field wherever text can be sent.
Example
Input
data
: 
My secret message!
