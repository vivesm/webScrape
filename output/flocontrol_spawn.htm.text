Call Flow
This function card enables you to run another flow within the context of your current flow. Your flow  resumes when the called flow finishes. 
If you don't need to  wait for the called  flow to finish, use the 
Call Flow Async
 function card instead.
Input
Field
Definition
Type
Required
Helper Flow
Opens a dialog so you can select the flow to execute within the context of your current flow.
Choose either an existing flow or create a flow.

The inputs that are defined on the helper flow.
