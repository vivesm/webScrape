Wait For
This function card pauses the flow and waits for a specific amount of time before resuming the flow.
The minimum wait time is one second and the maximum is 30 days. A value greater than 30 days results in the termination of the flow.
Input
Field
Definition
Type
Required
delay
The number of time units to pause the flow.
Number
TRUE
unit
Unit of time for the length of the pause.
These are the valid options:
second
minute
hour
day
Dropdown
TRUE
Output
This card has no output fields.
