テキスト埋め込みの生成
テキスト埋め込みは、テキストデータを数値ベクトルとして表現するための技術です。深層ニューラルネットワークを使用して、大量のテキストデータのパターンを学習し、テキストの意味と文脈を捉えたベクトル表現を生成します。これらのベクトルは、感情分析、テキスト分類、テキストの類似性など、さまざまな自然言語処理のタスクに使用されます。テキスト埋め込みの生成は、他の機械学習タスクの前段階として一般的に使用されます。詳細な情報は
こちら
(opens new window)
を参照してください。
テキスト埋め込みの生成アクション
#
入力
フィールド
説明
入力
埋め込みを取得するテキストを入力してください。入力は8192トークン（約6000単語）を超えてはいけません。
モデル
テキストを送信して埋め込みを取得するためのOpenAIモデルを選択してください。
ユーザー
OpenAIが監視および悪用の検出を支援するための、あなたのエンドユーザーを表す一意の識別子です。
#
出力
フィールド
説明
オブジェクト
データ値のオブジェクトのタイプです。通常、これは単一の値を持つリストです。
モデル
テキスト埋め込みに使用されるモデルです。
埋め込み
入力したテキストの埋め込みスコアを含むリストです。これは通常、別の機械学習モデルと組み合わせて使用されます。複数の入力が提供された場合、埋め込みの配列が返されます。
使用法
プロンプトトークン
プロンプトによって使用されるトークンの数です。
トータルトークン
プロンプトとレスポンスによって使用されるトークンの総数です。
