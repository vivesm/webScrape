#
Genies configuration
Genies are an Agentic feature available to customers with an active Workato subscription. Ensure that Genies are enabled in your account before you use this feature. Contact your Customer Success representative to enable Genies if you don't see the genies option in your workspace.
#
Core configuration requirements
Genies consist of the following core components:
Genie profile
Chat interface
AI model
Knowledge base
with at least one
knowledge recipe
Skills
with at least one skill recipe
CHAT INTERFACE AND AI MODEL CONFIGURATION
Genies can only be configured to use Slack or Microsoft Teams as the chat interface. Support for additional chat interface apps is in development. Additionally, the AI model can only be configured to use Workato AI. Support for additional LLMs is in development.

#
Genies configuration limits
Genies have the following configuration limits:
50 genies per Workspace
1 chat interface per genie
1 knowledge base per genie
1 AI LLM to power the genie’s core functionality
