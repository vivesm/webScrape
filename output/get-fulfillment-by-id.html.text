Shopifyコネクター - IDでフルフィルメントを取得する
このアクションは、ShopifyでIDでフルフィルメントを取得します。
#
入力
フィールド
説明
Fulfillment ID
フルフィルメントのIDを指定してください。
Order ID
注文のIDを指定してください。
#
出力
上記のアクションの出力は
Fulfillment
リソースです。詳細については、
Shopifyのドキュメント
(opens new window)
を参照してください。
