Cisco Webex Teams
#
Get room details action
This action retrieves the details of a specific room in Cisco Webex Teams.
#
Input fields
Field
Description
Room name
Select the room to retrieve details for. You can choose from a list of rooms or use a specific room ID.
#
Output fields
Field
Description
Creator ID
The ID of the person who created the room.
Room ID
The unique ID of the room.
Topic
The topic of the room.
Type
The type of the room.
Is locked
Indicates whether the room is locked.
Last activity
The timestamp of the last activity in the room.
Created
The timestamp when the room was created.
