Import data from a CSV file into a table
You can import data into a table from a CSV file.
Define the columns for a table.
Click the 
Import 
button to bring up the Import Data dialog.
Drag/drop a CSV file into the dialog or click 
Choose CSV File
.
Select a table column to which you want to map each CSV field. If the names match, that table column will be the default suggestion.
Click 
Import.
