#
Freshservice - List agent fields action
This action allows you to retrieve the details of a ticket based on the ticket ID.
#
Input
This action doesn't require any input fields.
#
Output
Output field
Description
Name
The internal field name used in the system.
Label for admins
The display label of the field when viewed by administrators.
Mandatory for admins
Indicates whether the field is required for administrators.
Editable by admins
Specifies if administrators can edit the field.
Label for agents
The display label of the field when viewed by agents.
Editable by agents
Specifies if agents can edit the field.
Mandatory for agents
Indicates whether the field is required for agents.
Visible to agents
Determines if agents can see the field.
Type
The type of field.
Default field
Indicates whether this is a default system field.
