#
Shopifyコネクター - 発送オーダー別の発送を一覧表示する
このアクションは、Shopifyインスタンス内の発送オーダーに関連するすべての発送を返します。詳細については、
Shopifyドキュメント
(opens new window)
を参照してください。オーダー、発送オーダー、および発送の関係について学ぶことができます。
#
入力
フィールド
説明
発送オーダーID
発送に関連する発送オーダーのIDです。
#
出力
上記のアクションの出力は、
Fulfillment
リソースのリストです。詳細については、
Shopifyドキュメント
(opens new window)
を参照してください。
