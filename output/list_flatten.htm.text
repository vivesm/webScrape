Flatten
Take a list that contains multiple lists and convert it into a single list.
Input
Field
Definition
Type
Required
lists
List of lists to be flattened.
Object
TRUE
Output
Field
Definition
Type
list
Flattened list.
Be sure to properly set the type of list.
In the examples below, the type for the first list is 
Number
, but the type for the second list is 
Text
.
