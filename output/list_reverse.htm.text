Reverse
Reverse the order of the items in a list.
Input Fields
list
(list)
: the list you want to reverse
Output Fields
reversed list
(list)
: the list in reverse order
Make sure that 
list
 and 
reversed list
 are both set to the same type.
Example:
If 
list
 is 
[1,2,3,4,5]
 then 
reversed list
 is 
[5,4,3,2,1]
