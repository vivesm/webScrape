Truncate
Cut a list down to a specified length.
This function can operate on lists of any type. To ensure this works correctly, you should make sure that the input and output lists are set to the same type.
Input Fields
list
: the input list
size
: the size you want to cut the list down to
Output Fields
truncated list
: a new list which is the original list cut down to the specified size.
Example
If 
list
 is 
[1,2,3,4,5]
 and 
size
 is 
3
 then 
truncated list
 will be 
[1,2,3]
