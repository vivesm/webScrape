Factorial
Calculate the factorial of a number, returning 0 if that number is less than 0. A factorial is the product of an integer and all the non-zero or non-negative integers less than it.
Input
Field
Definition
Type
Required
number
Numerical value whose factorial is calculated.
Number
TRUE
Output
Field
Definition
Type
output
Result of a factorial calculation for a specified value.
Number
Example
Factorial of 5 is 120 (
5 x 4 x 3 x 2 x 1
)
Factorial of 8 is 40320 (
8 x 7 x 6 x 5 x 4 x 3 x 2 x 1
)
