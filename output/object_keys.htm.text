Keys
The Keys card will generate a list of the keys within your object.
Input Fields
object
:  the object you want to extract keys from
Output Fields
output
: the keys of your object organized as a list of text
Examples
If input 
object
 is
{"Name":"Emily", "Age":"66", "Hair Color":"Brown", "Eye Color":"175"}
Then 
output
 is the list: 
["Name", "Age", "Hair Color", "Eye Color"]
