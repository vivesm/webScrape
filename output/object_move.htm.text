Move
Moves a value from one key to another key, which essentially renames the key while keeping the value the same.
Input Fields
object
:  the object to manipulate
source
:  the key in the original 
object
 containing the value you wish to rename
destination
:  the new key to move the value to
Output Fields
output
:  the new object with the renamed key-value pair now at the end of the object
Examples
If the input 
object
 is: 
{"one":"hello", "two":"goodbye"}
And 
source
 is 
one
 and 
destination
 is 
three
Then the 
output
 object will be: 
{"two":"goodbye", "three":"hello"}
