Values
The Values card will generate a list of the values within your object:
Input Fields
object
:  the object you want to extract values from
Output Fields
output
: the values of your object organized as a list. The list type can be selected based on the values within your object.
Examples
If input 
object
 is 
{"Name":"Emily", "Age":"66", "Hair Color":"Brown", "ID":"175"}
, then 
output
 is the list 
["Emily", "66", "Brown", "175"]
.
