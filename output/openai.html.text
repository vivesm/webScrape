#
OpenAIコネクター
OpenAI
(opens new window)
は、会話形式で対話する人気のあるChatGPTを開発している人工知能研究機関です。それに加えて、テキストの編集、画像の生成、分類など、さまざまなモデルを活用することができます。OpenAIを使用して内部的に独自のモデルを微調整し、公開することもできます。
#
APIバージョン
OpenAIコネクターは
OpenAI API
(opens new window)
を使用しています。
#
WorkatoでOpenAIに接続する方法
OpenAIコネクターは認証にAPIキーを使用します。OpenAIでAPIキーを取得するには、
こちら
(opens new window)
にアクセスしてください。APIキーは一度だけ表示されるため、安全に保存することをおすすめします。
