#
Recipe Copilot
HOW TO GET ACCESS
CopilotはすべてのWorkatoデータセンターのすべてのユーザーが利用できます。すべてのデータは米国のOpenAIのサーバーに送信されます。この機能にアクセスするには、
AI機能追加条項
(opens new window)
に同意する必要があることに注意してください。
WorkatoのRecipe Copilotは、大規模言語モデル（LLM）を活用して、レシピ作成体験を強化および加速します。このAI搭載機能は、すべての経験レベルのユーザーがレシピ作成プロセスを通じてガイドし、提案を提供し、タスクを自動補完して開発を迅速化します。
POWERED BY AI
Recipe Copilotで生成されたAI提案を展開前に正確性を確認し、テストしてください。
#
前提条件
Copilot
がワークスペースで有効になっていることを確認してください。Copilotを有効にするには、
Admin
システムロールが必要です。このロールが割り当てられていない場合は、ワークスペース管理者に連絡してこの機能を有効にしてください。詳細については、
Copilots
ドキュメントを参照してください。
PERMISSIONS
Copilotには詳細な権限設定はありません。ワークスペースで有効にすると、役割に関係なく、ワークスペース内のすべてのコラボレーターがアクセスできます。
#
Recipe Copilotの使い方
以下の手順に従って、Recipe Copilotを使用してタスクを自動化します：
1
Recipes > Create > Recipe
に移動します。
2
新しいレシピの名前を
Name
フィールドに入力します。
3
ドロップダウンメニューからプロジェクトまたはフォルダーを選択して、レシピの
Location
を選択します。
4
Start building
をクリックして、Recipe Copilotによるガイド付きレシピ作成プロセスを開始します。
5
この段階で、レシピの開始点を決定します：
ゼロから始めて独自のレシピを作成する。
Recipe Copilotを使用して、オートメーション目標に基づいてレシピ作成をガイドする。
Recipe Copilot start screen
Recipe Copilotは、
レシピを作成する
のに役立ちます。
#
Copilot生成の確認
Recipe CopilotのAIによる提案には不正確な点が含まれる場合があります。レシピの信頼性を確保するために、以下のガイドラインを確認してください：
#
提案の確認
AIが生成した各提案を確認し、要件を満たし、レシピのコンテキストに合っていることを確認します：
ロジックがオートメーション目標に合致していることを確認します。
提案されたデータフィールドや変換の正確性と関連性を確認します。
#
徹底的なテスト
本番環境での問題を避けるために、AIが生成した提案をテストします：
開発またはテスト環境でテストを実施し、ライブプロセスに影響を与えずに潜在的な問題を特定します。
実際の使用ケースをシミュレートするために現実的なテストデータを使用し、実際の条件下での機能を確認します。
さまざまなシナリオで提案が確実に機能することを確認するために、複数のテストを実行します。
