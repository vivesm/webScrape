#
Freshservice - Search agents action
The
Search agents
action returns results that match all of your search criteria. Returns a maximum of 30 records.

Input field
Description
Email
Provide the email ID of the agent.

Provide the work phone number of the agent.
Mobile phone number
Provide the mobile phone number of the agent.
Active
Select the boolean value from the drop-down menu.
State
Select the state from the drop-down menu.
