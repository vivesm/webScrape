#
UKG Pro - Terminated employee trigger
The
Terminated employee
trigger monitors UKG Pro for employee terminations. This trigger requires the connected UKG Pro account to have the
View
Web Service role.
#
Input
Input field
Description
Company
Select the company to monitor.
#
