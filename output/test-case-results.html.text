#
テストケースの実行結果の表示
Workatoがテストジョブの実行を完了すると、テストケースの実行結果に関する通知が届きます。
テストケースの実行結果モーダルには、実行された各テストケースに関する以下の情報が含まれています。
レシピ名
テストケース名
テストケースの実行結果（合格/不合格）
失敗したチェック（ある場合）
テストケース実行のジョブへのリンク
#
テストケースにチェックがない場合
ジョブの実行結果は、テストケースの実行結果と同じです。
#
テストケースの実行ジョブ
テストケースの実行が完了した後、
ジョブ
ページでレシピレベルでのすべてのテストケースの実行結果にアクセスできます。
リストからテストケース実行ジョブを選択する
#
テストケースジョブのメタデータ
各テストケースの実行に対するジョブページには、以下の情報が含まれています。
テストケースの実行結果を表示する
モックされたステップ
チェックがあるステップ
各チェックについて
チェックデータ
チェック結果
テストケースの実行結果
テストケース名とリンク
テストケースが削除された場合、Workatoは削除されたテストの名前のみを表示します。
