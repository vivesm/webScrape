Find Email
Find and return the first instance of a valid email address in a string.
Input
Field
Definition
Type
Required
text
Target of search for a properly formatted email address.
String
TRUE
Output
Field
Definition
Type
email
First instance of a properly formatted email address that is found in an input string. If the input value doesn't contain a valid email address, this output field remains empty.
String
Example 1
If the input 
text
 field contains 
You can reach me at fred@example.com or support@example.com
, the output 
email
 field will return 
fred@example.com
.
Example 2
For use cases where the validation of an email submission is required, you can use the Find Email function card in combination with 
Return Error
, 
Continue If
, 
If/Else
, or other function cards. You can use the Find Email function card to validate an email submission, and if the input value isn't a valid email address, an error will be thrown by a Return Error card.
