#
Tools
Workato provides a wide range of features to help you build recipes, integrate applications, and manage workflows. These features are divided into
Tools
and
Platform
capabilities, which provide different levels of functionality to enhance your automations within the Orchestrate platform.
#
Platform capabilities in Orchestrate
Platform capabilities provide advanced tools that enable you to manage integrations, events, and workflows, allowing you to extend and enhance your recipes. Availability of these features may vary based on your pricing plan. Refer to your pricing plan or contact your Customer Success Manager for more details.
Go to
Platform
in the Orchestrate sidebar to access these features.
Platform menu
The following tools are part of Workato’s Platform capabilities:
API platform
Turn your recipe or collection of recipes into API endpoints that external systems can interact with.
Learn more
.
Event streams
Enable recipes to subscribe to events from other recipes, reducing dependencies and improving scalability.
Learn more
.
Insights
Connect, manipulate, and visualize data within Workato’s no-code environment, enabling data-driven decisions and deeper insights into your workflows.
Learn more
.
Workbot
Build a chatbot for Slack or Microsoft Teams to automate workflows and engage users.
Learn more
.
Workflow apps portal
Add, group, and manage access to your portal to streamline access management and app provisioning.
Learn more
.
#
Tools in Orchestrate
Tools are focused on building, managing, and executing recipes. You can use Tools for foundational capabilities, such as recipe creation, data management, and human interaction.
Go to
Tools
in the Orchestrate sidebar to access these features.
Tools menu
The following tools are key features in Workato, though some may only be available in specific pricing plans. Refer to your pricing plan or contact your Customer Success Manager for more information.
Common data models
Create schemas for objects such as customers that can be used in multiple recipes.
Learn more
.
Connector SDK
Build custom application connectors to integrate with apps that aren’t natively supported by Workato.
Learn more
.
Custom Oauth profiles
Create and manage custom apps with control over branding, permission scopes, and OAuth profiles.
Learn more
.
Environment properties
Define account-wide variables that can be used across multiple recipes.
Learn more
.
FileStorage
Store and manage files within Workato. FileStorage allows you to create files from recipe data, retrieve stored files, and append content to existing files.
Learn more
.
Logs
Track and monitor recipe execution, errors, and performance.
Learn more
.
Lookup tables
Define lookup tables that your recipes can lookup to get additional information.
Learn more
.
Message templates
Create reusable message templates in formats such as HTML, JSON, or XML.
Learn more
.
On-prem groups
Group multiple on-prem agents into a common connection pool to distribute workload evenly and achieve high availability for recipes using on-prem systems.
Learn more
.
People task
Add human approval steps within workflows, requiring input before recipes can continue. Workato is in the process of deprecating the People task feature, and it may no longer be supported in future updates.
Learn more
.
Recipe lifecycle management
Manage the development, testing, and deployment of recipes through a structured process.
Learn more
.
