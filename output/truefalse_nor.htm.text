All False?
Return true if all inputs are false; otherwise false. Also known as the Boolean operator NOR.
Input Fields

: the first true/false value
input 2
: the second true/false value
click or drop here to create
: additional input fields that can be added
Output Fields
all false?
: a true or false value
true
: if all inputs are false
false
: if any input is true
