Decode Component
This function card decodes all URL-encoded characters from a string and outputs the decoded text.
This function is the inverse of the  
Encode Component
 function.
This card is useful to retrieve the value of a query parameter. If you need to decode a URL-encoded query string into a query object, use the 
Decode Query
 function card.
